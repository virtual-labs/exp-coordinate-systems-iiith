Instructions
  • This experiment is designed to teach how points and shapes are represented in computer graphics. Start by editing the coordinates of the point in the text boxes which are in the Point-Coordinates section.
  • Left click and drag the mouse to change the position of the point while observing the change in coordinates in the observations section.
  • Check the values in the text boxes by changing scaling factor.
  • Add new shapes by clicking on the “Add” button which provides us with different options to select the type of shape and also set its position. Multiple shapes may be added.
  • Double click on the object to select the shape for edit and delete. After double clicking on the object click on the edit button to edit the shape and similarly on the delete button to delete the selected shape.
  • Drag the vertices of the shapes from anywhere on the screen.
  • Tick on the checkboxes to add xy-grid and yz-grid and xz-grid and similarly untick to delete.
  • Enable Lock Vertices option and drag from anywhere on the canvas which results in translation of the whole simulation.
